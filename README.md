# iyo-scratch-blocks
WordPress Plugin for Visualizing Scratch Codes

## Usage
Write Scratch codes within `class="scratchblocks"`.
```php
<div class="scratchblocks">
...
</div>
```

## Notice
Please change `<` or `>` to `&lt;` or `&gt;` manually.
